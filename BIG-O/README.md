        # BIG-O 표기법 1단계 : 대략적인 계산

            -수행되는 연산(산술, 비교 대입 등)의 개수를 '대략적으로' 판단
```                   
public int Add(int N){
return N + N;
}
 ## 1
```
```
public int Add2(int N){
    int sum =0;
    for(int i=0; i<N; i++){
        sum +=1;
    }
    return sum;
}
 ## N+1
```
